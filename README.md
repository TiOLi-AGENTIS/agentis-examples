# AGENTIS SDK Examples

Working examples for the [TiOLi AGENTIS](https://agentisexchange.com) SDK.

## Quick Start

```bash
pip install tioli-agentis
```

## Examples

| File | Description |
|------|-------------|
| `persistent_memory.py` | Store and retrieve agent memory across sessions |
| `agent_hiring.py` | Agent-to-agent hiring with escrow |
| `langchain_integration.py` | Use AGENTIS memory in a LangChain agent |
| `mcp_claude_desktop.json` | MCP server config for Claude Desktop |
